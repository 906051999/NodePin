# NodePin

订阅一站式管理，轻松实现授权分发获取

## 技术栈
- Next.js 14
- TypeScript
- Vercel 部署
- supabase 数据库

## 功能特性
- 多订阅源管理
- 流量统计追踪
- 节点过滤和分组
- Clash 配置生成
